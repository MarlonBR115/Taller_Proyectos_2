# Otros Documentos

Esta carpeta conserva material historico o de apoyo generado en etapas anteriores del proyecto.

## Criterio de uso

- Si un documento de esta carpeta duplica contenido de `docs/sdd/`, se considera vigente la version de `docs/sdd/`.
- Si un documento contiene prompts o especificaciones antiguas, debe tratarse como antecedente y no como estado actual del sistema.
- No eliminar estos archivos sin validar que la informacion util este preservada en la documentacion principal.
