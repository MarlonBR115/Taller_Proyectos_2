## Supuestos y Notas Generales

- El costo por hora de mano de obra se fijó en **S/ 8.00/h** para un equipo de estudiantes universitarios.
- Las herramientas de desarrollo son **gratuitas o de código abierto** (VS Code, GitHub, Postman, React, Node.js, PostgreSQL).
- El servidor de despliegue utiliza el **plan básico de Railway o Render** (~S/ 37–38/mes).
- La contingencia del **10%** cubre imprevistos técnicos, cambios de alcance menores o gastos no previstos.
- El dominio web es **opcional**; puede omitirse si el sistema se despliega en subdominio gratuito.
- El proyecto **no considera gastos de viaje**, dado que el equipo trabaja de forma presencial/remota en el campus.
