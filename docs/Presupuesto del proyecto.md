# 1. Objetivo del Presupuesto

El presente documento estima los costos necesarios para el desarrollo e implementación del sistema de generación automática de horarios académicos utilizando técnicas de inteligencia artificial clásica y optimización mediante CSP/COP.

El presupuesto ha sido optimizado para minimizar gastos operativos y de infraestructura, utilizando herramientas gratuitas, software open source y servicios de hosting de bajo costo disponibles en Perú.

---

# 2. Supuestos Económicos

- El proyecto será desarrollado por un pequeño equipo técnico universitario.
- Se utilizarán herramientas gratuitas o de código abierto.
- El despliegue se realizará en un hosting económico tipo VPS compartido o hosting Node.js administrado.
- El desarrollo contempla:
  - frontend React
  - backend Node.js
  - motor CSP/COP
  - pruebas
  - documentación
  - despliegue básico
