# Documentacion de Seguimiento y Control

Esta carpeta agrupa documentos de monitoreo del proyecto, gestion de riesgos, impedimentos y estado de sprints. Su objetivo es mantener trazabilidad academica entre avances, riesgos, evidencias y decisiones del equipo.

## Registros de gestion

| Documento | Proposito | Estado |
|---|---|---|
| [Registro de Riesgos y Oportunidades](01_Registro_de_Riesgos_y_Oportunidades.md) | Identifica riesgos tecnicos, operativos y oportunidades de mejora para el prototipo de horarios. | Referencia principal para A004-03. |
| [KPI y Metricas](02_KPI_y_Metricas.md) | Define KPI implementados, parciales y propuestos para calidad de horarios y proceso academico. | Referencia principal de metricas de seguimiento. |
| [Trazabilidad Jira/GitHub](03_Trazabilidad_Jira_GitHub.md) | Relaciona tareas, archivos, evidencias, pruebas, metricas y riesgos. | Base de trazabilidad; PR y ramas pendientes si no existen. |
| [Registro de Impedimentos](02_Registro_de_Impedimentos.md) | Registra bloqueos que afectan avance o integracion. | Historico y vigente como seguimiento. |
