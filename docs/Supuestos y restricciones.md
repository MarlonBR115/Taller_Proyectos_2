# Suposiciones y Restricciones del Proyecto

## Suposiciones
* **Datos:** Se contará con datos académicos simulados o simplificados.
* **Disponibilidad:** El equipo tendrá disponibilidad para el desarrollo.
* **Herramientas:** Las herramientas tecnológicas serán suficientes.
* **Definición:** Las restricciones del problema estarán correctamente definidas.

---

## Restricciones
* **Infraestructura:** Disponibilidad de docentes, alumnos e infraestructura.
* **Créditos:** Límite de créditos (20–22).
* **Horarios:** No solapamiento de horarios.
* **Tiempo:** Tiempo limitado (hasta el 18 de junio de 2026).
* **Recursos:** Recursos técnicos limitados (equipos personales).
* **Complejidad:** Complejidad del problema de optimización.
* **Equipo:** Desarrollo realizado únicamente por estudiantes.
* **Alcance:** Alcance limitado a un prototipo funcional.
